# Engine notes:
@rendering folder
- asset manager: texture, shader, models hashmap with string (can have static shaders definitions cuz those can be used on bellow modules directly)
- scene: models, frustum, dirlight, skybox, visible models, camera
- forward_renderer: renderFullStage, renderVisibles, updateDirlightShadowmap
- shader
- skybox
- texture
- mesh: vector normals, vertices, indices, textures...
- model: meshs and material, load tinygltf
- material: pbr
- geometry: aabb, plane, frustum
- mesh_repo: plane, cube, triangle
- framebuffer
- light: directional (onlyu for now) -> shadowmap (framebuffer), direction, strength, color, shadowres

@platform
- window: glfw window stuff
- input: namespace only. input::isPressed input::isDown glm::vec2 input::getMouse input::_glfwCallback...

@on main.cpp
- load asset manager
- load model data/tree.gltf
- add plane on ground
- add things to scene
- window and input callbacks
- loop
- update inputs
- render scene
